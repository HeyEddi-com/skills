
# Vocabulary: UX flow auditor

- **Task flow:** named user job from product (e.g. update profile, invite teammate)
- **Click depth:** count of `fill` + `click` steps in trace
- **Click budget:** `max_clicks` in `.flow.json`: fail if exceeded
- **Friction:** extra navigation, hidden controls, errors, backtracks
- **Step actions:** `expect` (visible), `fill`, `click`
