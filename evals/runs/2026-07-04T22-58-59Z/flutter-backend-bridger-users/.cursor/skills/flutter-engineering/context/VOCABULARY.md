# Vocabulary — Flutter engineering

- **Stack config**: `.heyeddi/stack.json` — `"frontend": "flutter"`, `"backends": ["fastapi"]` | `["firebase"]` | both.
- **web_port**: Flutter web dev server (default **8085**).
- **api_port**: FastAPI dev server (default **8090**).
- **AppShell**: `lib/widgets/app_shell.dart` — drawer + scaffold chrome.
- **GoRouter**: path routes in `lib/router/app_router.dart`.
- **Riverpod**: default state management for new HeyEddi Flutter apps.
