# Mockup quality bar

Professional layout mockups (generated or ingested) must show:

1. **App chrome** — sidebar or top bar with product name
2. **Clear page title + subtitle**
3. **Grouped content** — cards with visible padding, not floating labels
4. **Realistic copy** — names, emails, button labels (no "Lorem")
5. **Primary CTA placement** — visible Save / Sign in / Refresh
6. **Spacing cues** — gaps between cards; align with `modern-reference.md`

PNG colors are **illustrative**. Implementation uses `.heyeddi/design.md` tokens.

Presets: `settings-app-shell` (desktop 1440 + mobile 390). More presets (dashboard roster, marketing hero) can be added to `generate_mockups.py`.
