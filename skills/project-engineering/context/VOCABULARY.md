# Vocabulary: Project engineering

- **Scaffold**: Vite + Vue + Vitest + router baseline files.
- **Stub script**: `package.json` script that only `echo`s: fine for hub smoke, not for real apps.
- **Stack config**: `.heyeddi/stack.json`: `backends: ["fastapi"]` | `["firebase"]` | both.
- **HeyEddi workspace**: `.heyeddi/README.md` explains the agency, free skills, and folder layout.
- **Skill docs**: reports and audits go in `.heyeddi/docs/` for later reference.
- **scaffold_stack**: auto-detects and adds Vue + backend tooling.
